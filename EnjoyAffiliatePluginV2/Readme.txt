The official Enjoy-Apartments.com affiliate plugin is a customizable plugin to add the Enjoy-Apartments Iframe or
banner to your website.

== Description ==

This Plugin creates an Iframe or Banner for Enjoy-Apartments.com Affiliate Partners to display on their website

If you are not an Affiliate partner yet you can join at: http://enjoy-apartments.com/en/affiliates-enjoy-apartments.html

Add the shortcodes you can find on the plugin settings page to your website and you're done!

== Features ==
* Iframe: Fill in your Affiliate details and show a booking Iframe on your website!
* Banner: Fill in your Affiliate ID and display different types of banners on your website!
* Easy to use: No code knowledge needed! 
* Multi-Language: Choose the Iframe or Banners Language.

== Installation ==

1. Download the zip file from this page 
2. Login to your WordPress dashboard 
3. Go to Plugins >> Add New, under Install Plugins title click on Upload and use the browse button to search the .zip file you have downloaded at point 1.
4. Once activated you can start using the plugin(you can find it in the main menu)
5. Use the settings page to customise the Iframe/Banner if needed. 


== Frequently Asked Questions ==

= With which WP version is the plugin compatible ? =

The plugin will work from WP ver 3.0 onwards.

= Can I place the banners on multiple positions ? =

Yes, you can use the shortcodes as much as you want.

= Can I use the plugin even if I'm not an affiliate yet? =

No. It's not possible to use the Iframe without being an affiliate,
however it is possible to use the banners without being an affiliate, but you could start earning commissions today by 
becomming an affiliate here: http://enjoy-apartments.com/en/affiliates-enjoy-apartments.html
